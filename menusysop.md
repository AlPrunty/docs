# //SYSOP Menu
***

These commands can be run while logged-on to the WWIV BBS. You will be prompted to enter the System Password when you use //SYSOP to access the SysOp Tools.

[A] Allow Editor  
[C] Chain Editor  
[E] Events Editor  
[I] Instance Editor  
[M] Menu Editor  
[U] User Editor  
[K] Print Votes  
[?] Display Help  
[B] Sub Editor  
[D] Dir Editor  
[G] Bulletin Editor  
[J] Conference Editor  
[T] Text Editor  
[V] Voting Editor  
[Q] Quit
